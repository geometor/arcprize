geometor.arcprize.solvers.gemini_logger

RST Documentation Generator for ARC Solution Sessions

Creates comprehensive documentation of the puzzle-solving process using ReStructuredText (RST) format, enabling detailed analysis and review.

Features:

  • Hierarchical session organization

  • Image handling and deduplication

  • Markdown conversion

  • Navigation link management

  • Token and timing metrics

  • Error logging

The logger creates a browsable documentation structure that captures the complete problem-solving process, including:

  • Grid states and transformations

  • Code execution results

  • Model interactions

  • Performance metrics
